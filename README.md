# Usagicar Theme Park

Three.jsで作られたうさぎ車のドライブゲームです。ユートピアな世界を走り抜けましょう。

## 遊び方

### 起動方法

```bash
git clone https://github.com/YOUR_USERNAME/usagi-car.git
cd usagi-car
npx serve
```

ターミナルに表示されるURL（例: `http://localhost:3000`）をブラウザで開きます。

### 操作方法

| キー | 動作 |
|------|------|
| ← / A | 左に90度曲がる |
| → / D | 右に90度曲がる |
| Space | 大ジャンプ（ふわっと着地） |
| F | 花火を発射 |
| B | バック走行に切り替え |

### マウス操作

- **ドラッグ**: 車を中心に視点を回転
- **スクロール**: ズームイン/アウト

## ゲーム内容

### ワールド

- 無限に広がる芝生と道路
- 曲がった先に新しい道が自動生成
- 木、建物、湖、花畑がランダムに配置
- カラフルな鳥が空を飛ぶ

### 敵

3種類の敵が道路付近に出現します：

1. **紫のトゲトゲ** - 多面体、ゆっくり回転
2. **赤い目玉** - 不気味に浮遊
3. **暗緑のキューブ** - 高速回転

敵にぶつかると画面が揺れます。花火で退治できます。

### 花火

- 進行方向に打ち上げ
- 縦方向に同心円状に爆発
- 各円は同じ色で構成
- 敵を倒す効果あり

## 技術仕様

- Three.js r160
- GLTFLoader（GLBモデル読み込み）
- RGBELoader（HDR環境マップ）
- OrbitControls（カメラ操作）

## ファイル構成

```
usagi-car/
├── index.html       # メインのゲームファイル
├── usagicar.glb     # うさぎ車の3Dモデル
└── README.md        # このファイル
```

## アニメーション

GLBモデルには以下のアニメーションが含まれています：

- Move_Entotsu (煙突の動き)
- Move_Head (頭の動き)
- Move_Jiku (軸の動き)
- Move_Fuusha (風車の動き)
- Go_Wheel (前進時の車輪)
- Back_Wheel (後退時の車輪)

ゲーム中は`Back`を含まないアニメーションが再生されます。
